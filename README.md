# \# HashedPassword (bcrypt demo)

# 

# \## Run locally

# 

# 1\. Create \& activate venv:

# &nbsp;  - `python -m venv venv`

# &nbsp;  - `.\\venv\\Scripts\\Activate.ps1` (PowerShell) or `source venv/Scripts/activate` (Git Bash)

# 

# 2\. Install deps:

# &nbsp;  - `pip install -r requirements.txt`

# 

# 3\. Run:

# &nbsp;  - `python app.py`

# 

# 4\. Visit:

# &nbsp;  - `http://127.0.0.1:5000/`

# 

# \## Notes

# \- This demo uses bcrypt for password hashing.

# \- `users\_sample.json` is safe to commit; real `users.json` with production data should be ignored.

#
